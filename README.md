## 📦 项目说明

本仓库为 **PagerMaid-Pyro v1.4.14** 的**非官方归档版本**，用于保留旧插件兼容性。  

由于自 v1.5.x 起，官方重构了核心架构，导致大量旧插件不再兼容，因此本项目用于：

- 📦 保留插件兼容的最后一个版本；
- 🧪 供开发者调试旧插件环境；
- 📁 提供完整源码供参考归档；
- ⛔ 本项目不接受新功能提交，纯归档用途。

---

## ⚠️ 插件兼容性提示

> **注意：大多数新插件已不再兼容 v1.4.14 及以前版本。**

- 内置的插件安装功能会默认拉取“最新版本插件”；
- 由于新插件基于重构后的新架构，**在本版本中将无法启用**，表现为插件“始终处于关闭状态”；
- 建议使用旧插件，或手动安装适配 v1.4.x 的早期插件版本。
